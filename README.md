### 🌎 How to use

1. Clone this repository
2. Download Yarn/NPM and VsCode
3. `$ cd vivonext`
4. `$ yarn install or npm install`
5. `$ yarn dev or npm dev`
6. Insert Your Phone Number
7. Choose a Ads
8. Choose a repeat value
9. Send
